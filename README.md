# Costify

## Project Overview
Costify is a project designed to help users manage their expenses efficiently. It provides tools for tracking spending, setting budgets, and generating reports to improve financial health.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/kevinalonsoquintino-design/Costify.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd Costify
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the application**:
   ```bash
   npm start
   ```
5. **Access the application** in your browser at [http://localhost:3000](http://localhost:3000).

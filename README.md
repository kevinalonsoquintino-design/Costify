# Costify

## Project Overview
Costify is a web application designed to provide users with tools to manage their expenses efficiently. The app allows users to track their spending habits, set budgets, and visualize their financial data.

## Tech Stack
- Frontend: React
- Backend: Node.js with Express
- Database: PostgreSQL

## Features
- User Authentication
- Expense Tracking
- Budget Management
- Data Visualization
- Reporting Tools

## Installation Instructions
1. Clone the repository: `git clone https://github.com/kevinalonsoquintino-design/Costify.git`
2. Navigate to the project directory: `cd Costify`
3. Install dependencies for both frontend and backend:
   - For the frontend: `npm install` in the `frontend` directory
   - For the backend: `npm install` in the `backend` directory
4. Set up the PostgreSQL database and add the connection details in the `.env` file.
5. Start the application:
   - For the frontend: `npm start` in the `frontend` directory
   - For the backend: `npm start` in the `backend` directory
